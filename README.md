# Graph Analyzer: Расчет времени вычисления замкнутых петель

## Назначение
Приложение для анализа направленных графов: генерации случайных графов, подсчета петель и измерения времени выполнения.

## Основные возможности
- Генерация случайных направленных графов с заданным количеством вершин и петель
- Графическое отображение графа с вершинами, направленными ребрами и петлями
- Подсчет замкнутых петель (self-loops) на основе адаптированной формулы Мейсона
- Точный хронометраж выполнения алгоритма
- Проверка корректности пользовательских данных

## Архитектура
Приложение построено по модульному принципу с четким разделением ответственности:

### Основные классы
- **GraphApp** - главное окно интерфейса
- **GraphModel** - модель данных графа (список смежности)
- **MasonFormula** - алгоритм поиска петель
- **GraphView** - компонент визуализации
- **GraphPanel** - панель для отображения графа
